Generar una distribución normal

Se usa rnorm para generar una distribución normal:

r = “random”, norm = “normal”

n = ¿cuantas observaciones?,

mean = ¿cual es el promedio?

sd = ¿cual es la desviación estándar?

Visualizar los datos

La distribución se puede visualizar con “geom_density”

my_data1 |> 
  ggplot( aes( x = CI) )+ # aes = aesthetic -> estéstica
  geom_density()
my_data2 |> 
  ggplot(aes(x = CI))+
  geom_density()
my_data1 |> 
  ggplot(aes(x = CI))+
  geom_histogram()
`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.
hist(my_data1$CI)
my_data1 |> 
  ggplot(aes(CI))+
  geom_histogram(binwidth = 10)
